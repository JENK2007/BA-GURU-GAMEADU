# BA GURU GAME ADU 🎮

Welcome to the **Ba Guru Game Adu** repository! This is a hyper-modern, cyberpunk-inspired arcade landing page designed with a premium, futuristic aesthetic.

## 🚀 Features

* **Hyper-Modern Design:** Uses glassmorphism, glowing neon borders, and dynamic gradients for a stunning visual experience.
* **Fluid Typography:** Features a curated gaming font stack (`Orbitron`, `Rajdhani`, `Share Tech Mono`) that scales perfectly across all screen sizes (from mobile to 4K desktop) using fluid `clamp()` sizing.
* **Interactive UI:** Includes a custom cursor follower, hover animations, and a fully responsive slide-out mobile drawer.
* **Next.js & Tailwind CSS:** Built on modern, fast web technologies for optimal performance and developer experience.

## 🛠️ Tech Stack

* **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Language:** TypeScript
* **Fonts:** Google Fonts (Orbitron, Rajdhani, Share Tech Mono)

## 💻 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/JENK2007/BA-GURU-GAMEADU.git
   cd BA-GURU-GAMEADU
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **View the site:**
   Open [http://localhost:3000](http://localhost:3000) (or whichever port Next.js provides) in your browser.
